# Habits Tracker

Programa de hábitos feito durante a Next Level Week de uma aplicação para tracking de hábitos para web e mobile. AS tecnologias utilizadas estão presentes abaixo.

- NodeJS
- TypeScript
- Fastify
- Prisma
- Vite
- React
- React Native
- TailwindCSS
- CLSX
- DayJS
- Axios
- Zod
- Expo
